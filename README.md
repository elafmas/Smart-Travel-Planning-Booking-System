# Smart Travel Planning & Booking System ✈️

## Overview

Smart Travel Planning & Booking System is a Java-based application developed as a Data Structures course project. The system manages travel destinations, hotels, and customer reservations while demonstrating the practical implementation of fundamental data structures.

## Features

* Manage tourist and business destinations
* Store and manage hotel information
* Create and process customer reservations
* Search destinations by city
* Display business destinations
* Find the most expensive hotel
* View and manage the reservation queue
* Calculate total revenue

## Technologies

* Java
* Object-Oriented Programming (OOP)
* Data Structures

## Data Structures Used

* **Singly Linked List** – Used to store destinations and hotels with dynamic memory allocation, allowing efficient insertion and searching.
* **Queue (Linked List Implementation)** – Used to manage customer reservations following the FIFO (First-In, First-Out) principle for fair booking processing.

## Project Structure

* `TravelPlace`
* `TouristPlace`
* `BusinessPlace`
* `Hotel`
* `Reservation`
* `SinglyLinkedList`
* `LLQueue`
* `Main`

## Concepts Demonstrated

* Singly Linked List implementation
* Queue implementation using Linked List
* Searching and filtering data
* Dynamic data storage
* FIFO reservation processing
* Object-Oriented Programming (Inheritance, Encapsulation, and Polymorphism)
